# FireReport AI

Converts plain English incident descriptions into NFIRS 5.0 compliant reports using AI.

## Overview

FireReport AI is a web-based tool designed to help fire service personnel quickly generate standardized NFIRS 5.0 incident reports from natural language descriptions. By leveraging AI, officers can describe an incident in plain English and receive a structured report with properly coded fields.

## Features

- **Natural Language Input** - Describe incidents in plain English
- **AI-Powered Extraction** - Automatically extracts and codes NFIRS 5.0 fields
- **Mobile-Friendly** - Works seamlessly on tablets and phones in the field
- **Copy to Clipboard** - Easy export of generated reports
- **Code Compliance** - Ensures reports meet NFIRS 5.0 standards

## NFIRS Fields Supported

- Incident Type Code
- Property Use Code
- Structure Type
- Area of Fire Origin
- Extent of Damage (flame, heat, smoke)
- Apparatus Used
- Personnel Assigned
- Injuries/Deaths
- Property Loss Estimates
- Narrative Summary

## Getting Started

1. Open `index.html` in a web browser
2. Paste or type your incident description
3. Click "Generate Report"
4. Review the structured NFIRS fields
5. Copy to clipboard and export as needed
