# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-19

### Added
- Initial release of FireReport AI
- Dark theme UI for web application
- Anthropic API integration for AI-powered incident report generation
- NFIRS 5.0 field extraction from plain English incident descriptions
- Support for major incident fields: incident type, property use, structure type, damage extent, apparatus, personnel, injuries, and property loss
- Mobile-friendly responsive design
- Simple copy-to-clipboard functionality for generated reports
